# 清除手機瀏覽器緩存指南

## iPhone (Safari)

### 方法 1：清除 Safari 緩存
1. 打開「設定」→「Safari」
2. 向下滾動找到「清除瀏覽記錄和網站資料」
3. 點擊「清除瀏覽記錄和網站資料」
4. 確認清除

### 方法 2：清除特定網站資料
1. 打開「設定」→「Safari」→「進階」→「網站資料」
2. 搜尋你的網站（例如：kaichen-designer.github.io）
3. 向左滑動網站名稱
4. 點擊「刪除」

### 方法 3：強制重新載入（最快）
1. 在 Safari 中打開網站
2. 長按「重新整理」按鈕（圓形箭頭）
3. 選擇「重新載入而不使用快取內容」

## Android (Chrome)

### 方法 1：清除 Chrome 緩存
1. 打開 Chrome 瀏覽器
2. 點擊右上角「⋮」選單
3. 選擇「設定」→「隱私權和安全性」
4. 點擊「清除瀏覽資料」
5. 選擇「快取圖片和檔案」
6. 點擊「清除資料」

### 方法 2：清除特定網站資料
1. 在 Chrome 中打開網站
2. 點擊網址列左側的「鎖定」圖示
3. 選擇「網站設定」
4. 點擊「清除並重設」

### 方法 3：強制重新載入
1. 在 Chrome 中打開網站
2. 點擊右上角「⋮」選單
3. 選擇「重新載入」或長按重新載入按鈕

## 通用方法（所有瀏覽器）

### 無痕模式測試
1. 打開瀏覽器的無痕/隱私模式
2. 訪問網站
3. 這樣可以確保使用最新的代碼，不受緩存影響

### 添加版本參數（開發者方法）
在網址後面添加 `?v=2` 或時間戳，例如：
```
https://kaichen-designer.github.io/forest-sound/?v=2
```

## 推薦步驟

1. **先嘗試強制重新載入**（最快）
2. 如果還是有問題，**清除特定網站資料**
3. 如果還是不行，**清除整個瀏覽器緩存**
4. 最後可以嘗試**無痕模式**測試
